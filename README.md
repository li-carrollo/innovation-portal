# innovation-portal

A centralized hub where users can access various tools, projects, or platforms.

## Files

- `index.html` contains the portal layout
- `assets/sit-plus-logo.png` contains the Service IT+ logo used in the header
- `styles.css` contains the visual design and responsive behavior
- `script.js` contains the editable project list and full-card click behavior

## Update project links

Edit the `projects` array in `script.js` and replace the sample `https://example.com/...` URLs with your real innovation project links.
